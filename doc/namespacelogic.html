baueraerger is the function for the "Bauer ärgere dich Karte".

It allows the player to install a "Bauer ärger dich nicht" into his "Stall" preventing the "Bauer schrubbt die Sau" karte from working

Parameters
schwein_nrindex of Schwein which the card should be used on
gamedirectory

The function needs to meet a few conditions to allow the card to be used:

  • It needs to check if there is a "Stall" installed to build a "Bauer ärger dich"
  • It needs to check if there is not aready a "Bauer ärger dich Karte" installed
  • It needs to check if the selected Schwein exists
Returns
If all are fulfilled it returns a True statement else it returns a False statement and the according ERROR message onto the board_log.
